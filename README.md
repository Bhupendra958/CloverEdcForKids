# 🌱 CloverEduForKids – Interactive Learning Platform for Kids

CloverEduForKids is an interactive educational website designed to make learning fun, engaging, and accessible for children. The platform provides subject-based content, quizzes, stories, and interactive activities to enhance knowledge and critical thinking skills.

🌐 **Live Website:** http://cloverkids123.infinityfree.me

---

## 🚀 Features

- ✅ Subject-based Learning (Math, English, Science, Social Science, Physical Education)
- ✅ Interactive Quizzes with Timer
- ✅ Multiple Difficulty Levels (Easy, Medium, Hard)
- ✅ Moral Story Section
- ✅ Educational Video Section
- ✅ Clean and Modern UI
- ✅ Fully Responsive (Mobile + Tablet + Desktop)
- ✅ Dynamic Functionality using JavaScript

---

## 🖥️ Website Modules

### 🏠 Home Page
Provides an overview of the platform and available subjects.

### 📚 Subjects Section
Organized learning materials for different subjects.

- Timed quiz functionality  
- Multiple levels

Short stories with moral values to encourage positive thinking.

Subject-based educational video integration.

## ⚙️ Installation (Local Setup)

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/clovereduforkids.git
   ```

2. Open the folder in VS Code.

3. Run using Live Server (for frontend-only version).

4. If backend is used:
   - Import the database into phpMyAdmin.
   - Update database credentials in `config.php`.
   - Run using XAMPP or any local server.
